# 💸 SpendWise AI

SpendWise AI is a modern personal finance and expense tracking desktop application built using Python, SQLite, and CustomTkinter.

## ✨ Features
- Add and delete expenses
- Monthly expense tracking
- Budget warning alerts
- Category-wise analytics chart
- CSV report export
- AI spending insights
- Search expenses by category
- Interactive modern UI

## 🛠 Technologies Used
- Python
- CustomTkinter
- SQLite
- Pandas
- Matplotlib

## ▶ How to Run
Install required libraries:

pip install -r requirements.txt

Run the app:

python main.py
